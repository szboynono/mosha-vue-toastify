Mosha Vue Toastify ​

A light weight and fun Vue 3 toast or notification or snack bar.

Vue Version Support: This library supports Vue 3.0+ and is tested with Vue 3.5.x. For the best experience, we recommend using Vue 3.5 or higher.

Features ​

  • Super easy to setup! try follow this
  • Swipe to close
  • Support for Composition API
  • Written in typescript, full typescript support
  • Super light weight
  • Define behavior per toast
  • Fun progress bar to display remaining time
  • A lot more coming!
